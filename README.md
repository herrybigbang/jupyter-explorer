- **Mac/Linux**: Use `unzip filename.zip`  

### **Step 3: Launch Jupyter**  
1. Open **Terminal/Command Prompt**  
2. Navigate to the extracted folder:  
   ```bash

   ```
3. Run:  
   ```bash
   jupyter notebook
   ```
4. Your browser will open automatically at `http://localhost:8888` 🎉
